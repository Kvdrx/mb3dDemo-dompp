# mb3dDemo-dompp


This is a demo of the tests written on the Mb3d website with the DOM-PP library


Installing and building DOM-PP WITH MB3D 
----------------------------------------

## Git Clone

```bash
git clone https://github.com/Kvdrx/mb3dDemo-dompp.git
```

## Install

```bash
npm install  
```

## Testing

```bash
npm test
```
